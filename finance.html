Najprej nastavitev

Da se aplikacija lahko poveže s tvojim Google Drive-om, potrebuje OAuth Client ID. To je enkratna nastavitev — sledi vodiču in prilepi sem.

  1. Pojdi v Google Cloud Console in odpri svoj projekt
  2. Pod APIs & Services → Credentials ustvari OAuth 2.0 Client ID (vrsta: Web application)
  3. V Authorized JavaScript origins dodaj URL te aplikacije (npr. https://uporabnik.github.io)
  4. Prilepi Client ID spodaj in shrani

Podroben vodič po korakih: glej priložen dokument Navodila za nastavitev.md
